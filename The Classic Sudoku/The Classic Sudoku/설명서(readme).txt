/* 설명서 (readme) */
[ 해당 문서는 The Classic Sudoku에 관련된 파일 및 폴더에 대한 설명을
  포함합니다. ]


1. 문제지 / 답안지 저장 관련 (Saves 폴더)

* The Classic Sudoku 폴더의 Save폴더(이하 폴더)에 내가 풀었던 
  문제지와 답안지를 저장하는 기능이 있습니다.

* 게임 내부의 설정으로 게임에서 나왔던, 문제지와 답안지의 저장 
  여부를 설정할 수 있습니다.
  ( 게임설정 > 문제지 저장 > 문제지 저장 ON 항목 )

* 저장된 게임은 .txt(텍스트) 확장자 형식으로 저장됩니다.

* 저장된 문제지(혹은 문제지 + 답안지)는 저장 순서대로 번호가 
  매겨져서 저장됩니다.


2. 음악 재생 관련 (Sound 폴더)

* The Classic Sudoku 폴더의 Sound폴더(이하 폴더) 안에 아무런 파일이
  없을시엔 게임 설정을 하더라도 음악이 재생되지 않습니다.
  ( 음악 파일이 없는데 재생설정을 하게되면, 경고음이 반복적으로
    출력됩니다. )

* 폴더내부에 들어가야 하는 파일 이름은 다음 3가지와 같습니다.
  ( k_pop.wav / pop_song.wav / bgm.wav ) 
  이외의 파일이 들어가도 무관하나, 설정의 적용을 받지 못합니다.

* 만약 음악 플레이리스트를 바꾸고 싶다면, 음악 파일의 이름을 
  위 3가지 형식에 맞추어 바꾼 후, 넣어주면 됩니다.
  ( 단, 음악 파일 확장자는 반드시 .wav 여야만 합니다. mp3확장자나
    기타 확장자는 .wav로 인코딩후 넣어줘야 적용이 가능합니다. )


3. 게임 데이터 관련 (gameData.bin 파일)

* The Classic Sudoku 폴더내부에 생성되는 gameData.bin 파일은 
  게임에서 사용되었던 데이터를 저장하는 파일입니다.

* 게임 내부의 데이터 ( 전적, 레벨, 티어, 기타 옵션 등)을 초기화
  하고 싶다면 gameData.bin을 삭제를 해주시면 됩니다.

* gameData.bin 파일은 삭제가 되어도 프로그램 시작시 기본 값으로
  재생성됩니다.
